# Frontend (Studio Suite)

Panel React + TypeScript dla domeny salonowej.

## Kluczowe sekcje UI
- Dashboard
- Kalendarz wizyt
- Kartoteka klientow
- Zasoby (salony/pracownicy)
- Uslugi
- Pakiety
- Kolory/produkty
- Raporty
- Uzytkownicy

## Uruchomienie lokalne
```bash
cd frontend
npm install
npm run dev
```

App: `http://localhost:5173` (dev) lub `http://localhost:8082` (docker)

## Build
```bash
npm run build
```
