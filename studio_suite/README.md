# Studio Suite

Panel i API dla domeny salonowej.

## Moduly aktywne

- Auth i role (`admin`, `manager`, `employee`)
- Salony i pracownicy (`/resources`)
- Produkty, uslugi i pakiety (`/legacy/catalog`)
- Raporty legacy (`/legacy/reports`)

## Moduly usuniete

- Wyswietlacze
- Dzwonki
- Klienty urzadzen (Python i Android)
- Downloady APK/EXE

## Struktura

```text
studio_suite/
  backend/
  frontend/
  docs/
  scripts/
```
