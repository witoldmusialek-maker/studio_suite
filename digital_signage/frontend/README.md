# Frontend - Digital Signage

Panel zarządzania systemem digital signage oparty na React + TypeScript.

## Struktura

```
frontend/
├── src/
│   ├── components/      # Komponenty React
│   ├── pages/          # Strony
│   ├── contexts/       # Context API
│   ├── services/       # Serwisy (API, WebSocket)
│   ├── types/          # Typy TypeScript
│   ├── theme.ts        # Konfiguracja MUI theme
│   ├── App.tsx         # Główny komponent
│   └── main.tsx        # Entry point
├── package.json
├── tsconfig.json
└── vite.config.ts
```

## Instalacja

```bash
cd frontend
npm install
```

## Uruchomienie

```bash
npm run dev
```

Aplikacja dostępna pod: http://localhost:3000

## Build

```bash
npm run build
```
